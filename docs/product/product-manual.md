# 脉刻 (Maik) 产品手册

## 1. 产品定位

**脉刻** — 新一代社交化招聘平台，连接求职者、招聘者与人才伯乐三方生态。

### 核心价值
- 求职者：精准匹配，快速入职
- 招聘者：高效触达优质候选人
- 人才伯乐：人脉变现，推荐有奖

---

## 2. 角色体系

### 2.1 求职者 (JobSeeker)
**定位**：找工作的人

**核心能力**：
- 创建/上传简历
- 浏览/搜索职位
- 投递简历
- 查看投递进度
- 接收面试邀请
- 与招聘者/伯乐沟通

**开通方式**：注册即自动开通

---

### 2.2 招聘者 (Recruiter)
**定位**：发布职位、管理招聘流程的 HR/用人经理

**核心能力**：
- 创建/管理公司信息
- 发布/编辑/下架职位
- 查看收到的简历
- 管理候选人流程（初筛→面试→offer）
- 设置职位悬赏/推荐奖金
- 与求职者/伯乐沟通

**开通方式**：
- 个人注册后申请企业认证
- 或受企业管理员邀请加入

---

### 2.3 猎头 (Headhunter)
**定位**：专业全职猎头，深度服务高端人才招聘

**核心能力**：
- 所有人才伯乐能力
- 专属猎头后台
- 高级人才库管理
- 企业深度合作权限
- 更高比例佣金

**开通方式**：
- 平台邀请制
- 或提交资质审核通过

---

### 2.4 人才伯乐 (TalentScout)
**定位**：普通用户可开通的轻量级推荐功能

**核心能力**：
- 建立个人人才库
- 推荐朋友简历给招聘者
- 追踪推荐进度
- 成功入职后获得推荐奖金

**开通方式**：
- 任何注册用户可在设置中一键开通
- 无需资质审核
- 可随时关闭

**与猎头的区别**：
| 维度 | 人才伯乐 | 猎头 |
|------|----------|------|
| 开通门槛 | 无 | 资质审核/邀请 |
| 服务深度 | 轻量推荐 | 全流程服务 |
| 佣金比例 | 标准比例 | 更高比例 |
| 人才库规模 | 个人人脉 | 专业积累 |
| 企业合作 | 无 | 有 |

## 3. 核心流程

### 3.1 求职者流程

```
注册/登录
    ↓
完善简历（手动填写/上传解析）
    ↓
浏览职位 / 接收推荐
    ↓
投递简历
    ↓
查看进度（已投递→已查看→面试→offer）
    ↓
入职确认
```

### 3.2 招聘者流程

```
注册/登录
    ↓
企业认证
    ↓
发布职位（可设置悬赏金额）
    ↓
接收简历（主动投递 + 伯乐推荐）
    ↓
筛选候选人
    ↓
面试流程管理
    ↓
发放 offer
    ↓
入职确认 → 发放悬赏
```

### 3.3 人才伯乐流程

```
开通伯乐身份
    ↓
添加人才到个人库（朋友简历）
    ↓
浏览职位，匹配推荐
    ↓
向招聘者推荐人才
    ↓
追踪推荐状态
    ↓
人才入职 → 获得推荐奖金
```

## 4. 功能模块

### 4.1 用户系统 (User)
- 注册/登录（手机号/微信/邮箱）
- 实名认证
- 角色管理（求职者/招聘者/伯乐身份切换）
- 账号设置

### 4.2 简历系统 (Resume)
- 简历创建/编辑
- 简历解析（PDF/Word/图片 → 结构化数据）
- 简历搜索（Meilisearch）
- 隐私设置（公开/仅投递/隐藏）

### 4.3 职位系统 (Job)
- 职位发布/编辑/下架
- 职位搜索/筛选
- 职位收藏
- 职位推荐算法

### 4.4 投递系统 (Application)
- 简历投递
- 投递状态追踪
- 面试邀请
- offer 管理

### 4.5 人才库系统 (TalentPool)
- 个人人才库（伯乐）
- 企业人才库（招聘者）
- 人才标签/分组
- 人才跟进记录

### 4.6 推荐系统 (Referral)
- 人才推荐
- 推荐进度追踪
- 悬赏/奖金计算
- 结算记录

### 4.7 消息系统 (Message)
- 站内信
- 面试通知
- 系统通知
- 微信/短信推送

### 4.8 企业系统 (Company)
- 企业信息
- 成员管理
- 招聘数据看板
- 账单/悬赏管理

## 6. 商业模式

### 收入来源
1. **企业会员费** — 高级招聘功能
2. **悬赏抽成** — 成功入职后平台抽成 10-20%
3. **简历解析服务** — 按量计费
4. **广告位** — 职位/企业推广

### 成本结构
- 服务器/云服务
- 简历解析 API
- 短信/推送服务
- 支付通道手续费
