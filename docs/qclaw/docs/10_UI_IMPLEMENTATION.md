## 一、设计系统（必须严格遵守）

### 颜色令牌
```css
:root {
  /* 背景 */
  --bg-primary: #FFFFFF;
  --bg-secondary: #F5F5F7;
  --bg-tertiary: #F0F0F2;
  
  /* 文字 */
  --text-primary: #1D1D1F;
  --text-secondary: #86868B;
  --text-muted: #A1A1A6;
  
  /* 强调 */
  --accent: #0071E3;
  --accent-hover: #0077ED;
  --accent-light: rgba(0,113,227,0.08);
  
  /* 状态 */
  --success: #34C759;
  --warning: #FF9500;
  --error: #FF3B30;
  
  /* 边框 */
  --border: #E5E5E7;
  --border-light: #F2F2F7;
  
  /* 阴影 */
  --shadow-card: 0 1px 3px rgba(0,0,0,0.04), 0 1px 2px rgba(0,0,0,0.02);
  --shadow-modal: 0 20px 40px rgba(0,0,0,0.08);
}
```

### 字体
```css
--font-body: 'PingFang SC', 'Noto Sans SC', -apple-system, BlinkMacSystemFont, sans-serif;
--font-mono: 'SF Mono', monospace;
```

### 字号层级
```
text-3xl: 30px / font-bold     — 页面大标题
text-2xl: 24px / font-semibold — 区块标题
text-xl: 20px / font-semibold  — 卡片标题
text-lg: 17px / font-medium    — 重要正文
text-base: 15px / font-normal  — 标准正文
text-sm: 13px / font-normal    — 次要信息
text-xs: 11px / font-normal    — 标签/角标
```

### 圆角规格
```
rounded-sm: 8px   — 按钮、输入框
rounded-lg: 16px  — 卡片、弹窗
rounded-full: 9999px — 标签、头像
```

## 六、AI Agent 实现指令

### 通用约束
- 所有颜色使用CSS变量，不写死色值
- 所有间距使用Tailwind间距系统
- 所有圆角使用定义的三规格
- 响应式优先移动端（RN App）
- Web后台优先桌面端（md:以上）

### 组件实现顺序
1. 全局样式定义（Tailwind config覆盖）
2. 基础组件（Button, Input, Card, Badge）
3. 布局组件（Header, Sidebar, TabBar）
4. 业务页面（按优先级）

### Prompt模板（给v0.dev）
